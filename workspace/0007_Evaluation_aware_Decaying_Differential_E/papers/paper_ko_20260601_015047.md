# Abstract
본 논문은 지역 에너지 커뮤니티의 가격 결정 문제에서 사회적 효용과 공정성(예: Jain 지수, 최저 수혜자 보장)을 동시에 최적화하기 위한 새로운 진화적 최적화 기법, Evaluation-aware Decaying Differential Evolution (EA-DDE)를 제안한다. EA-DDE는 (1) 탐색→활용 전환을 유도하는 세대별 감쇠(decay) 스케줄로 DE의 변이·교차 강도를 제어하고, (2) 세대 내부에서 cheap→expanded→full 평가를 반복하는 evaluation-aware racing(진행형 샘플링)과 surrogate-guided 예산 배분, 랜덤 감사(audit)를 결합하여 고비용·확률적 시뮬레이션 평가를 효율화한다. 공정성 제약은 라그랑지안으로 다루며 primal–dual 업데이트로 λ를 조정한다. 알고리즘의 의사코드, 오차·false-negative 제어를 위한 통계적 샘플링 규격, 복잡도 분석 및 상세 재현성 계획을 제시하고, 제안기법의 유효성을 검증하기 위한 실험 설계(시나리오·기준·비교군·평가 지표)를 기술한다.

# 1. Introduction
지역 에너지 커뮤니티(LEC; Local Energy Communities)는 분산형 발전(태양광 등)과 유연한 수요 관리를 통해 지역 내 에너지 거래·가격 결정을 수행한다. 가격 정책은 전체 사회적 효용(social welfare)을 극대화하면서도, 특정 집단이나 취약 계층이 과도한 불이익을 받지 않도록 공정성을 확보해야 한다. 실제 운영에서는 재생에너지 변동성·수요 불확실성·시뮬레이션 기반 비용평가로 인해 각 후보 가격정책의 평가 비용이 높고 잡음이 크다. 따라서 제한된 평가 예산 내에서 효율적으로 우수한(사회적 효용·공정성 관점의) 해를 찾아야 하는 실용적 어려움이 존재한다.

기여
- EA-DDE: DE 프레임워크에 세대별 감쇠 스케줄과 세대 내 evaluation-aware racing(cheap→expanded→full 평가)을 결합한 최초의 통합 알고리즘을 제안한다. 이로써 고비용·확률적 평가 환경에서 평가 예산을 효율적으로 배분하며 탐색·활용의 전환을 제어한다.
- 공정성 제약의 라그랑지안 처리: 사회적 효용을 목적함수로, 공정성 지표(g(x))를 제약으로 취급하고 primal–dual 온라인 업데이트(λ_t)로 제약 강도를 조정하여 실시간 트레이드오프를 관리한다.
- 이론적·실무적 안전장치: Hoeffding/Chernoff 기반 신뢰구간을 통한 조기 제거(early elimination)의 false-negative 확률 제어 규격과 랜덤 감사 확률(p_audit), surrogate 불확실성 고려 예산 배분 규칙을 제시한다.
- 재현 가능성: 의사코드·하이퍼파라미터·복잡도·실험 프로토콜(데이터셋·시나리오·평가 지표·통계검정)을 상세히 제공한다.

구성: 관련 연구(Section 2), 방법론(Section 3), 실험 설계(Section 4), 결론(Section 5).

# 2. Related Work
- Differential Evolution(1995–1997)과 적응형 변형: DE는 연속 최적화에 널리 쓰여왔으며, jDE, SaDE 등은 F와 CR을 적응적으로 조정한다. 본 논문은 DE 기반이지만 탐색→활용 전환을 명시적 감쇠 스케줄로 제어한다.
- Multi-objective / Fairness-aware Optimization: 공정성을 다목적 최적화 혹은 제약으로 처리한 연구들이 존재한다. 본 연구는 공정성 지표를 제약으로 간주하고 라그랑지안 기반의 온라인 업데이트를 결합한다.
- 고비용 평가 환경의 샘플링 및 레이싱: F-Race, successive halving, Hyperband 등은 후보 평가를 단계적으로 확장해 예산을 절감한다. 본 논문은 DE 내부 세대단위에서 이러한 레이싱을 통합하고, surrogate 예측·불확실성을 활용해 예산을 배분한다.
- Surrogate-assisted Evolutionary Optimization: 계산 비용이 높은 평가 문제에 surrogate 모델(GP, RF 등)을 결합하는 연구가 많다. EA-DDE는 surrogate의 불확실성(예: predictive variance)을 고려한 예산 할당 규칙을 제안한다.
- 통계적 안전성: Hoeffding/Chernoff 불평등을 활용한 조기 종료와 false-negative 제어에 대한 이론적 근거를 차용한다.

차별점: (1) DE 내부의 세대-내 평가 스케줄링과 변이 파라미터 감쇠를 통합, (2) 공정성 제약을 라그랑지안으로 온라인 제어, (3) 통계적 false-negative 제어를 포함한 실무적 안전장치의 결합.

# 3. Method
목표 문제
- 결정변수 x ∈ X ⊂ R^d: 지역가격 정책 파라미터(예: 시간대별 요율, 퇴직 보조금 파라미터 등).
- 목적: maximize f(x) := E_ω[W(x; ω)] (사회적 효용의 기댓값, ω는 불확실성 샘플)
- 제약: g_i(x) ≤ c_i (i = 1..m), 예: fairness(x) ≥ φ0를 g(x) = φ0 - fairness(x) ≤ 0 로 표현
- 평가가 고비용이고 확률적: 하나의 full evaluation은 N_sims 시뮬레이션 샘플에 기반.

알고리즘 개요
- 기본: DE 세대 루프(t = 0..T-1), 인구(population) 크기 NP.
- 세대 내부는 K 단계의 evaluation-aware racing: 단계 k에서 각 후보에 대해 n_k 샘플(cheap: 작음 → expanded → full: 큼)으로 성능 추정치 μ̂_k 및 신뢰구간 CI_k을 계산. 라운드마다 불량 후보를 통계적으로 조기 제거.
- Surrogate: 각 후보의 μ̂_k와 함께 surrogate(예: Gaussian Process 또는 Random Forest)를 학습/업데이트하여 후보들의 예측 평균 μ̃ 및 예측 불확실성 σ̃를 제공. 예측 불확실성이 큰 후보에 추가 예산 배분.
- 라그랑지안 처리: 목적 L(x, λ) = -μ̃(x) + λ * max{0, ĝ(x)} (부호는 maximize 전환 고려). λ 업데이트: λ_{t+1} = max(0, λ_t + η_t * (ĝ_pop^+ - ρ)), where ĝ_pop^+는 세대 내 선택된 후보(또는 인구 평균)의 제약 위반량의 추정치, ρ는 허용 한계(보정값). η_t는 스텝사이즈(감쇠 가능).
- 탐색→활용 감쇠: mutation factor F_t, crossover CR_t, 및 DE 변형 확률 p_mut(t)은 감쇠 스케줄로 조정:
  F_t = F_min + (F_0 - F_min) * exp(-α * t / T)
  CR_t = CR_max - (CR_max - CR_min) * exp(-β * t / T)
  여기서 α, β ≥ 0은 감쇠 속도. 초기에는 큰 F·다양한 CR로 탐색, 후반에는 작게 하여 수렴 촉진.
- 감사(audit): 조기 제거 후보 중 일정 비율 p_audit을 랜덤 선택해 full evaluation으로 재검증하여 false-negative 억제.

의사코드 (요약)
입력: NP, T, K, n_1<...<n_K (샘플 크기 단계), total eval budget B, F_0, F_min, CR_min, CR_max, α, β, η_0, p_audit, δ (신뢰수준)
초기화: population P0 (size NP), λ0 = 0, surrogate S (빈)
for t = 0..T-1:
  compute F_t, CR_t by decay schedule
  generate offspring Q via DE mutation/crossover (F_t, CR_t)
  C = P_t ∪ Q
  for k = 1..K:
    for each candidate x in C not yet eliminated:
      run n_k - n_{k-1} 추가 샘플 (n_0 := 0) 시뮬레이션 → 업데이트 샘플 평균 μ̂_k(x), 분산 s^2_k(x)
      compute CI_k(x) (Hoeffding/Chernoff 기반 또는 t-interval)
    eliminate 후보 x if CI_k 상한 < 상위(톱) 후보의 CI_k 하한 by margin (통계적 기준, δ 분배)
    update surrogate S using 현재 샘플 데이터 (μ̂_k 등)
    allocate 추가 예산 우선순위: 높은 μ̃ + 높은 σ̃ 또는 현재 CI 폭이 큰 후보 우선
  perform random audits: for eliminated candidates with prob p_audit, do full evaluation to detect false-negative; if audit 재수행 결과 유의하면 복원
  select next generation P_{t+1} from surviving candidates using Lagrangian score: score(x) = μ̃(x) - λ_t * max{0, ĝ(x)}
  update λ_{t+1} = max(0, λ_t + η_t * (ĝ_sel^+ - ρ))
endfor
출력: 최종 후보 집합(비교 Pareto front 후보들)

통계적 조기제거 규칙과 false-negative 제어
- 각 단계 k에서 후보 x의 평균 성능 μ̂_k(x)은 독립 샘플의 평균이라 가정(혹은 교차검증 샘플 분리).
- Hoeffding 불평등로부터, 샘플 수 n이면 P(|μ̂ - μ| ≥ ε) ≤ 2 exp(-2 n ε^2 / (b-a)^2), 구간 [a,b]가 성능 범위. 목표 false-negative 허용 δ'에 대해 필요한 n ≥ ( (b-a)^2 / (2 ε^2) ) ln(2/δ').
- 조기 제거 판단: 후보 x를 y보다 불리하다고 제거하려면 μ̂_k(x) + CI_upper(x) < μ̂_k(y) - CI_lower(y) - Δ_margin 으로 설정. Δ_margin은 안전 마진. 전체 단계 및 전체 비교에 대해 union bound를 적용하여 δ 전체를 제어(δ_total = Σ 단계·비교 δ_k).
- 감사 확률 p_audit은 이상적 audit 수 A = -ln(δ_FN) / ln(1/(1 - r)), 여기서 r는 제거된 후보 비율 근사. 실무적으로 p_audit을 작게 잡아도 false-negative 확률을 줄이는 경험적 보완.

라그랑지안 λ 업데이트 (구체 규칙)
- λ_{t+1} = max(0, λ_t + η_t * (ḡ_t^+ - ρ))
  ḡ_t^+ = mean_{x∈P_{t+1}} max(0, ĝ(x)) 또는 최악 후보의 ĝ
  η_t = η_0 / sqrt(t+1) (감쇠 스텝사이즈 권장)
- 해석: 제약 위반이 크면 λ 증가로 제약 준수를 강제, 위반이 작으면 λ 감소 가능.

Surrogate·불확실성 활용
- surrogate는 후보 (x, μ̂, s^2) 쌍을 학습. 예: Gaussian Process(저차원) 또는 Random Forest(고차원).
- 예산 배분 우선순위 score_eval(x) = μ̃(x) + κ * σ̃(x) + γ * CI_width(x), κ, γ ≥ 0. 높은 불확실성 후보에 추가 샘플 배분.

복잡도·확장성
- 한 세대 당 최악 평가비용: O(NP * n_K) 시뮬레이션. 실제로 racing으로 조기제거돼 평균 비용 << NP * n_K.
- surrogate 학습: GP O(N^3) (데이터 수 N_samp), RF O(N_samp log N_samp). 실무적으로 RF나 sparse-GP 권장.
- 병렬성: 각 후보의 시뮬레이션은 독립적이므로 병렬 분산 가능. surrogate 업데이트 주기를 늘려 통신 비용 저감 가능.

실현적 하이퍼파라미터(권장 초기값)
- NP = 50, T = 100, K = 3, n = [5, 20, 100] (cheap→full), B ~ NP*T*avg(n_k) 제한
- F_0 = 0.8, F_min = 0.2, CR_min = 0.1, CR_max = 0.9, α = 5, β = 5
- η_0 = 0.1, p_audit = 0.05, δ_total = 0.05, κ = 1.0, γ = 0.5
- 위 값은 문제 특성에 따라 조정. 민감도 분석 권장.

# 4. Experiments
목적: 제한된 평가 예산 하에서 EA-DDE가 (1) 사회적 효용과 (2) 공정성 지표(예: Jain index, min utility)를 동시에 개선하는지를 검증하고, (3) 평가 예산 효율성과 false-negative 제어 효과를 평가한다.

데이터셋·시나리오
- 합성 시나리오: 다양한 규모(소형 20가구, 중형 100가구, 대형 500가구)와 PV 보급률, 수요 탄력성 분포를 갖는 에너지 커뮤니티 시뮬레이터. 각 개체의 유틸리티 함수와 가격 반응을 가정(예: 선형 수요 탄력성 + 랜덤성).
- 실제/반실제 시나리오: 공개된 에너지 소비 데이터(예: REFIT, UK-DALE, Pecan Street)를 기반으로 시간대별 수요 재구성 및 PV 생산 합성.
- 불확실성 모델링: PV 생산의 계절·기상 기반 랜덤 샘플, 수요의 주간·일일 변동성, 시뮬레이션 내 무작위 시드로 평가 잡음 도입.

비교군 (baselines)
- DE (기본), jDE, SaDE
- MODE: NSGA-II 또는 MODE 변형 (다목적 최적화로 사회적 효용·공정성 동시 최적화)
- Surrogate-assisted DE (기본 surrogate 없이 레이싱 없음)
- Successive Halving / Hyperband 적용된 DE (세대 외부에서 레이싱)
- Bayesian Optimization (BO) 기반 최적화(고비용 설정에서의 표준 비교군)

평가 지표
- 사회적 효용: 평균 총 복지 (평균 및 분포)
- 공정성: Jain's fairness index, min utility (최저 수혜자 유틸리티)
- Pareto front 근사성: hypervolume(사회적 효용 vs fairness)
- 평가 효율성: 총 시뮬레이션 횟수 대비 성능
- false-negative rate: 감사로 검출된 조기제거 오류 비율
- 계산 시간(총, surrogate 학습 포함)

실험 프로토콜
- 랜덤 시드 30개 반복, 각 반복에서 동일한 초기화와 시뮬레이션 샘플 사용(공정 비교)
- 예산별(낮음·중간·높음) 실험: B_low = 0.2 * baseline, B_med = baseline, B_high = 2 * baseline
- 통계검정: 성능 지표에 대해 Wilcoxon signed-rank test (p<0.05) 및 bootstrap 95% CI
- Ablation: (A) EA-DDE 전체, (B) EA-DDE without racing, (C) EA-DDE without surrogate, (D) EA-DDE without decay (고정 F, CR), (E) EA-DDE with no audit
- 민감도: α, β, p_audit, η_0, K, n_k 조정

예상 검증 포인트
- 제한된 평가 예산에서 EA-DDE는 동등 예산의 기본 DE·SaDE 대비 높은 평균 사회적 효용 및 개선된 공정성 지표를 달성할 것으로 기대한다(특히 불확실성·잡음이 큰 환경에서).
- racing과 surrogate의 결합은 동일 예산 하에 더 빠른 초기 수렴과 좋은 Pareto 근사를 제공할 것으로 예상된다.
- audit과 신뢰구간 기반 제거 규칙은 false-negative 비율을 통제하면서 평가 예산을 절감한다.

재현성
- 코드 공개(깃허브), 시뮬레이터·시나리오 스크립트 포함. 실험 설정(하이퍼파라미터·시드) 명시. surrogate 라이브러리(예: scikit-learn, GPy) 버전 명시.

# 5. Conclusion
본 논문은 고비용·확률적 평가 환경에서 지역 에너지 커뮤니티의 가격정책 설계 문제를 위해 EA-DDE를 제안하였다. EA-DDE는 DE의 탐색·활용 전환을 감쇠 스케줄로 제어하고, 세대 내 evaluation-aware racing·surrogate-guided 예산 분배·라그랑지안 공정성 제어·감사 메커니즘을 통합한다. 이론적 신뢰구간 기반의 false-negative 제어 규격과 현실적인 하이퍼파라미터·복잡도 분석을 제공하며, 제한된 평가 예산 하에서 사회적 효용과 공정성의 동시 개선을 목표로 하는 실험 설계도 제시했다. 향후 연구로는 대규모 분산 실행(엣지 병렬화), 온라인 실시간 가격 업데이트와의 통합, 그리고 실제 LEC 필드 실증 적용이 있다.

Abstract
We propose Evaluation-aware Decaying Differential Evolution (EA-DDE) for pricing optimization in Local Energy Communities, jointly optimizing social welfare and fairness. EA-DDE integrates a generation-wise decay schedule to steer DE from exploration to exploitation, and an intra-generation evaluation-aware racing scheme (cheap→expanded→full evaluations) with surrogate-guided budget allocation and random audits to handle costly, stochastic simulations. Fairness constraints are handled via a Lagrangian primal–dual update for λ. We present pseudocode, statistical sampling rules to control false negatives, complexity analysis, and a reproducible experimental plan.

1. Introduction
Local Energy Communities require pricing policies that maximize social welfare and ensure fairness under uncertain renewable generation and demand; evaluations are typically costly and noisy. We contribute (1) EA-DDE combining decay schedules and intra-generation racing within DE; (2) Lagrangian online control of fairness constraints; (3) statistical safety mechanisms (Hoeffding/Chernoff-based CI, audits); (4) full reproducibility details.

2. Related Work
Covers DE and adaptive variants (jDE, SaDE), fairness-aware optimization, surrogate-assisted evolutionary algorithms, racing methods (F-Race, successive halving), and statistical bounds.

3. Method
Defines the optimization problem, algorithmic components: DE with decaying F_t/CR_t, intra-generation K-stage racing with sample counts n_k, surrogate predictions μ̃ and uncertainty σ̃, Lagrangian update λ_{t+1} = max(0, λ_t + η_t (ḡ_t^+ - ρ)), audit probability p_audit, and statistical sample-size formula from Hoeffding for controlling false-negative probability. Pseudocode and complexity (per-generation worst-case O(NP*n_K) evaluations; surrogate overhead) and recommended hyperparameters are given.

4. Experiments
Design: synthetic and real consumption datasets, various community sizes, baselines (DE, jDE, SaDE, MODE, surrogate DE, successive halving, BO), metrics (social welfare, Jain index, min utility, hypervolume, evaluation efficiency, false-negative rate), protocol (30 seeds, budget levels, Wilcoxon tests), ablations, sensitivity analyses, and reproducibility notes.

5. Conclusion
EA-DDE offers an integrated, practical approach for fairness-aware pricing under costly stochastic evaluations, with theoretical and practical safety measures. Future work: large-scale parallelization and real-world deployment.
